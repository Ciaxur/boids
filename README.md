# Boids
Implementation of the boids algorithm ([Boids - Wiki](https://en.wikipedia.org/wiki/Boids)).

# Usage
Run `make -j69 run` to compile and run the project.

## Keybinds:
  - `Space`: Toggles outlines like avoid/align radii and velocity vector.
